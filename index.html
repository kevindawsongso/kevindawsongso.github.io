Lazarus Group Detection Lab

This guide details a step-by-step process for building a cybersecurity home lab using VMware Workstation Pro. Its core purpose is to enable simulating and detecting the tactics, techniques, and procedures (TTPs) of the Lazarus Group, a notorious North Korean APT focused on financial gain and espionage. The guide includes comprehensive configuration steps, decision-making insights, simulation techniques, challenges, and troubleshooting. It is organized into phases: infrastructure setup, security tool implementation, and attack simulation/analysis, providing practical skills for threat detection.

Cloud Security & Honeypots

Driven by a desire to gain hands-on cloud security experience and understand real-world threats, I built a lab with a T-Pot honeypot on AWS EC2 to capture attack patterns. Integrating GuardDuty, Detective, VPC Flow Logs, and CloudWatch allowed me to practice advanced detection and block malicious traffic via WAF. Segmenting assets in separate VPCs reinforced secure cloud design principles. Analyzing attacks like RDP/SSH brute-force significantly enhanced my knowledge and refined incident response tactics.
